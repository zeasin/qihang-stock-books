## 一个包裹从进仓到出仓，系统里发生了什么？

我们用一批货从入库到发给消费者的完整流程，来看看系统是怎么运转的：

### 第一步：货到了，先入库 📦

货主的货送到仓库，你只需要在系统里做三件事：

**收货** → **质检** → **上架**

系统会自动推荐最佳库位。比如矿泉水这种重货放底层货架，小件轻货放上层。上架完成后，库存实时更新，货主在自己的端口就能看到货已经进仓了。

### 第二步：货在仓里，随时清楚 📊

货在仓库里不是一放了之。你可以：

- **随时查库存**：哪个商品在哪个库位、有多少，一目了然
- **设置库存预警**：低于安全库存自动提醒，不怕断货
- **定期盘点**：用PDA扫码盘点，不用拿着纸质单子来回跑
- **批次管理**：哪些货先到期，系统帮你盯着

### 第三步：订单来了，开始拣货 🛒

消费者的订单通过API同步到系统后，会发生什么？

系统会把多个订单**智能合单成波次**，拣货员拿着**PDA**按最优路线一次搞定一批订单的拣货——不用来回跑。

拣完货还要**复核打包**，确保没错才能进入下一步。

### 第四步：打包发货，自动出单 📮

打包完成后，系统自动对接物流渠道生成面单，发货后物流轨迹自动回传。

如果是跨境包裹，系统还能帮你生成报关所需的资料，不用手动填表。

### 第五步：费用结算，不再头疼 💰

这一步是很多云仓最头疼的——**算钱**。

系统支持各种计费规则：
- 按件收费、按重量收费、按体积收费
- 按天收仓储费
- 入库费、出库费、打包费、物流费、增值服务费

月底一键生成账单，货主对账清清楚楚，再也不用自己拉Excel算了。
